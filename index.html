Work

My senior project involved working with my colleages with a company named Whirlpool who specializes in smart appliances. Using React-Native we helped create a interal user application for Whirlpool employees to contribute to data collecting from annotating pictures of varies items.

School project using Apple's GameKit framework using Swift 5. With GameKit we can create applications that allow us to interact with other iOS users through Game Center's network. Some of the many examples you can implement to your own application are: achievements, leaderboards, and challenges. Through Game Center you are also able to add real time multiplayer or turn based multiplayer. We intergrated a simple rock paper scissors game using Gamekit.

East Asian Cuisine is a personal project I helped create for my Mom in her Asian food side business. It is a pretty simple website with the main feature being a menu with pictures and descriptions of each item.
