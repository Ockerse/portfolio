The social media website is a web application that allows users to interact with their friends and share their thoughts, photos, and videos. It is built using HTML, CSS, JavaScript, MySQL, and PHP. The website has several features, including a homepage with friends' posts, a way to add and chat with friends, customize your profile by changing your profile picture, and a register and login screen. The homepage of the website displays the latest posts from the user's friends. Users can view, like, and comment on the posts. They can also share their own posts by uploading photos or text updates. To add friends, users can search for other users on the website and send friend requests. Once the request is accepted, they can chat with their friends using the messaging feature. Users can also customize their profiles by changing their profile picture and updating their bio. The backend of the website is managed with phpMyAdmin, which is a popular database management tool. It allows the website to store user information and post data in a MySQL database. The website also has a register and login screen to ensure that only authorized users can access the site. Users can create an account by providing their name, email address, and a password. The login screen checks the user's credentials against the database and allows access to the website. Overall, the social media website provides a platform for users to connect with their friends and share their thoughts and experiences. Its features, such as the messaging system, friend requests, and profile customization, make it a fun and engaging platform for users to interact with each other.

The Simon Says game is a classic memory game that has been recreated using modern web technologies, such as HTML, CSS, and JavaScript. The game features a series of buttons that light up and produce sounds in a specific pattern, and the player must repeat the pattern correctly to progress to the next level. The game includes multiple levels, each with increasing difficulty and complexity. As the player progresses, the game speeds up, and the pattern becomes longer, challenging the player's memory and reaction time. In addition to its engaging gameplay, the Simon Says game also features sound and animation effects, making the game more immersive and enjoyable to play. The sound effects play whenever the player clicks a button or gets the sequence wrong, adding to the overall excitement of the game. The animation effects, on the other hand, make the game more visually appealing by adding a cool animation to each button when it is pressed. Overall, the Simon Says game is a fun and challenging project that showcases the power of HTML, CSS, and JavaScript when used together. The project's features, including level progression, sound effects, and animation effects, make it an excellent example of what can be achieved with these web technologies.

Tindog is a website designed for dog owners who are looking to connect their furry friends with other dogs in the area. The website is modeled after Tinder, with a similar swiping interface that allows users to browse through profiles of other dogs and their owners. The homepage features a clean and modern design, with bold text and eye-catching images of happy dogs. Users can easily create a profile for their own dog, complete with photos, a bio, and basic information such as breed, age, and location. They can then browse through profiles of other dogs in the area, swiping right to show interest and left to pass. The site also includes a messaging feature, allowing users to chat with each other and set up playdates for their dogs. There is also a section for events and meetups, where users can post details about local dog-friendly events and activities. Overall, Tindog provides a fun and interactive way for dog owners to connect with each other and foster new friendships for their furry companions.
